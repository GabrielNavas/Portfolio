# Meu Portfólio Pessoal

Bem-vindo ao repositório do meu portfólio! Este projeto foi desenvolvido para centralizar minhas experiências, projetos e habilidades técnicas de uma forma visual e interativa.

 **Acesse o portfólio** -> [Gabriel Portfolio](https://portfolio-pi-sable-62.vercel.app/)

---

## Tecnologias Utilizadas

Para a construção deste projeto, utilizei as seguintes tecnologias:

* **HTML5 & CSS3:** Estruturação e estilização responsiva.
* **JavaScript:** Lógica de interface e interatividade.
* **Design:** Foco em uma interface limpa (Clean UI) e experiência do usuário (UX).

---

## Sobre o Projeto

O objetivo deste portfólio é apresentar quem eu sou como desenvolvedor. Ele contém seções dedicadas a:

1.  **Apresentação:** Uma breve introdução sobre minha carreira e interesses.
2.  **Habilidades:** Tecnologias que domino.
3.  **Projetos:** Galeria com meus principais trabalhos e links para os repositórios.
4.  **Contato:** Links diretos para LinkedIn, E-mail e outras redes sociais.

---

## Como rodar o projeto localmente

Se você deseja clonar este repositório para estudar a estrutura ou fazer modificações:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/GabrielNavas/Portfolio.git](https://github.com/GabrielNavas/Portfolio.git)
    ```

2.  **Navegue até a pasta:**
    ```bash
    cd Portfolio
    ```

3.  **Abra o arquivo `index.html`** no seu navegador de preferência.
